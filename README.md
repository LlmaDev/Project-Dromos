# Project-Dromos

A web application for supply chain management and optimization, focusing on evaluating the impact of trade tariffs on suppliers, routes, and products.

## Overview

This application enables simulation of import/export scenarios, helping companies identify the most cost-effective suppliers and routes by considering associated tariffs and providing tools for cost calculation and optimization.

## Technologies

- **Backend**: Java + Spring Boot
- **Frontend**: React + TypeScript + Vite
- **Database**: PostgreSQL (H2 for testing)
- **Testing**: JUnit + Mockito (backend), Jest (frontend)
- **CI/CD**: Jenkins

# Dromos Tariff Calculator - Frontend

Uma aplicação React TypeScript simples para calcular tarifas de importação usando a API do backend Dromos.

## Tecnologias Utilizadas

- React 18
- TypeScript
- Tailwind CSS
- Axios
- Vite

## Pré-requisitos

- Node.js (versão 16 ou superior)
- npm ou yarn
- Backend Dromos rodando na porta 8080

## Instalação e Execução

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse a aplicação em: `http://localhost:3001`

## Build para Produção

```bash
npm run build
```
